# SafeNest/Conest - Project Overview

## Purpose
A React Native mobile application with Node.js backend helping single parents find safe, verified, and compatible roommates for shared housing. The platform prioritizes **child safety** while making housing affordable through cost-sharing.

## Core Mission
- Target users: Single parents (ages 25-45, income $25K-75K)
- Problem: Currently paying 40-60% of income on rent
- Solution: Safe roommate matching with mandatory verification
- Safety-first: NO child data storage, mandatory background checks

## Business Model
- Freemium: Basic matching free, premium $4.99/month
- Success fee: $29 one-time when roommates sign lease
- B2B partnerships with affordable housing providers
- Eligible for housing innovation grants

## Key Success Metrics
- Zero child safety incidents
- 95% user verification rate
- <24hr background check completion  
- 80% successful housing matches
- Average housing stability: >12 months
