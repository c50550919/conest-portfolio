## 🔧 Maintenance Commands

**Reset all passwords**:
```bash
node -e "const bcrypt = require('bcrypt'); bcrypt.hash('Test1234', 10).then(h => console.log(h));"
# Copy hash and run:
psql -c "UPDATE users SET password_hash = '<hash>' WHERE email LIKE '%@test.com';"
```

**Regenerate profile photos**:
All photos are generated dynamically via URL, no storage needed!

**Check profile completeness**:
```sql
SELECT
  u.email,
  p.profile_completed,
  p.profile_completion_percentage,
  p.bio IS NOT NULL as has_bio,
  p.profile_photo_url IS NOT NULL as has_photo
FROM users u
JOIN parents p ON u.id = p.user_id
WHERE u.email LIKE '%@test.com';
```
