## Archived Files

### SUCCESS_FEE_FRAUD_PREVENTION.md
**Date Archived**: 2025-01-11
**Reason**: Tiered success fee model ($0/$15/$29) was rejected due to:
1. **Gaming Risk**: Off-platform transactions make success fees unenforceable
2. **Privacy Violation**: Lease verification requires invasive document collection
3. **Enforcement Cost**: Manual review overhead exceeds potential revenue
4. **Market Misalignment**: Success fees only work when transactions flow through platform (like Airbnb)

**Replacement**: Simple pricing model:
- $39 verification (one-time)
- $14.99 premium subscription (monthly)
- $99 bundle (verification + 6mo premium)

**Decision Date**: 2025-01-11
**Decision Maker**: Product team
**Related Issue**: Pricing model sustainability analysis (based on real provider costs: Veriff $1.39 + Certn $28.50 = $29.89)
